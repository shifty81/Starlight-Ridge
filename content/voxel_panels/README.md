# Voxel Panel Kits

Phase 53f introduces data-first voxel-pixel panel kits.

These `.ron` files are editable in the native egui **Assets → Voxel Panels** tab. A kit contains:

- palette/material entries
- modular pieces
- per-piece voxel cells
- sockets for future assembly/snapping
- bake targets for later sprite/collision/runtime output

This phase is intentionally limited to 2D depth-slice editing and validation. Full 3D preview, socket assembly, and bake execution are planned for follow-up phases.
