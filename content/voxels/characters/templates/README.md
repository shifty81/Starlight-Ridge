# Phase 53e High-Density Character Templates

These `.vox` files are first real high-density pixel-voxel base templates for Starlight Ridge.

Rules:

- Bald base heads.
- Clean-shaven base faces.
- Neutral bodies.
- No baked identity traits.
- No final clothing.
- Modular slots/anchors are marked with guide colors.
- Feet-center pivot marker is included.
- Future hair, facial hair, hats, clothing, gloves, boots, tools, and accessories should be separate modular overlays.

These are still template bases, not final player/NPC art.
