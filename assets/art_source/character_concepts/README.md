# Uploaded Human Character Concepts

These four uploaded PNGs are integrated as first-party art-source references for the Starlight Ridge human character direction.

They are concept/reference material for the human male and human female base character style. They are not yet final runtime spritesheets; the runtime paperdoll and mannequin assets still live under `assets/textures/characters/`.

## Files

- `uploaded_human_concept_01.png` — original upload `34266456.PNG`, 308x455.
- `uploaded_human_concept_02.png` — original upload `456456.PNG`, 313x463.
- `uploaded_human_concept_03.png` — original upload `111.PNG`, 152x393.
- `uploaded_human_concept_04.png` — original upload `124132515.PNG`, 173x385.

## Intended use

- Guide the human male / human female base sprite direction.
- Inform Character Studio paperdoll proportions, clothing overlays, hair, boots, shirts, pants, hats, backpack/rig overlays, and tool-use silhouettes.
- Keep source references separate from runtime sprites so the current game build remains stable.
