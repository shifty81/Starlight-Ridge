# Character Template QA Checklist

Use this checklist before promoting any generated/refined character base.

## Identity restrictions

- [ ] No hair.
- [ ] No facial hair.
- [ ] No final outfit.
- [ ] No baked identity features.
- [ ] Neutral face only.
- [ ] Neutral body only.

## Shape quality

- [ ] Head silhouette reads from front.
- [ ] Head silhouette reads from side.
- [ ] Neck transition is clean.
- [ ] Shoulders are defined.
- [ ] Torso has chest/waist/hip rhythm.
- [ ] Arms are not stick-like.
- [ ] Elbows are visible enough for rigging.
- [ ] Hands have enough volume for tool grips.
- [ ] Legs have knee landmarks.
- [ ] Feet are stable and readable.
- [ ] Back silhouette is clean.

## Technical

- [ ] Feet-center pivot marker exists.
- [ ] Right-hand grip marker exists.
- [ ] Left-hand grip marker exists.
- [ ] Hair/hat anchor exists.
- [ ] Face overlay anchor exists.
- [ ] Back item anchor exists.
- [ ] Dimensions match profile.
- [ ] Voxel count is within offline budget.
- [ ] File opens in MagicaVoxel.
- [ ] File validates as `.vox`.

## Editor workflow

- [ ] Shows in Voxel Generator / Generated Assets.
- [ ] Can be opened in MagicaVoxel.
- [ ] Can be opened in Blender/Blockbench workflow if converted/exported.
- [ ] Can be used by Phase 53f rig preview.
