# Pixel-Voxel Generator External Tool Bridge

## MagicaVoxel

Primary cleanup and `.vox` authoring tool. The editor should open generated files directly in MagicaVoxel, then refresh after save.

## Blockbench

Optional pose, attachment, and rig planning tool. It should not replace canonical `.vox` source assets.

## Blender

Bake/render automation tool for thumbnails, turntables, directional sprite sheets, and future normal/depth maps.

Phase 53b only adds bridge profiles and placeholder commands. Full automation belongs to later phases.
