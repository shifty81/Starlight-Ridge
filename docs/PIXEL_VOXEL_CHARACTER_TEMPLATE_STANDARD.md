# Pixel-Voxel Character Template Standard

## Target dimensions

```txt
Body A: 56 x 36 x 96
Body B: 62 x 38 x 104
Future hero detail: 64-72 x 40-48 x 112-128
```

## Required anatomical landmarks

- bald head mass
- neutral face plane
- jaw/chin blockout
- ears
- neck
- shoulders
- chest/ribcage
- waist taper
- hips/pelvis
- upper/lower arms
- elbow landmarks
- hand blockouts
- upper/lower legs
- knee landmarks
- foot/boot blockouts

## Modular slots

```txt
hair
hat
face_overlay
shirt
jacket
pants
boots
gloves
right_hand_tool
left_hand_tool
back_item
accessory_front
accessory_back
```

Base templates must never bake hair or facial hair into the body.
