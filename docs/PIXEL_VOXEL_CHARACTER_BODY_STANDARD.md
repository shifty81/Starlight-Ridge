# Pixel-Voxel Character Body Standard

## Character base standard

Starlight Ridge character bases should be high-density pixel-voxel mannequins, not chunky block placeholders.

## Required base rules

```txt
No hair.
No facial hair.
No baked identity traits.
No final outfit.
Neutral face base only.
Neutral body base only.
```

Hair, facial hair, hats, outfits, gloves, boots, backpacks, tools, and accessories must be modular overlays.

## Recommended dimensions

```txt
Body A:
  56 × 36 × 96

Body B:
  62 × 38 × 104

Player neutral:
  60 × 38 × 104

NPC average:
  56 × 36 × 96
```

## Required anatomical landmarks

Every base body should clearly show:

- head mass,
- jaw/chin plane,
- neutral face plane,
- neck,
- shoulder caps,
- ribcage/chest,
- waist taper,
- hips/pelvis,
- upper arms,
- elbows,
- forearms,
- hands,
- upper legs,
- knees,
- lower legs,
- feet.

## Readability rules

- Silhouette must read from front, side, and back.
- Hands must be large enough for tool grip markers.
- Feet must define a stable feet-center pivot.
- Head must remain bald and clean.
- Face landmarks must not look like facial hair.
- Body should be neutral enough for player, NPC, and outfit overlays.
- Pixel-voxel density should support form, not random surface noise.

## Template marker policy

Guide markers are allowed in template files as long as they are documented.

Markers should be removable during final bake or hidden by material/profile settings.
