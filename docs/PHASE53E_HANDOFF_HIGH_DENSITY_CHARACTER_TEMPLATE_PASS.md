# Phase 53e Handoff — First Real High-Density Character Template Pass

Phase 53e should rebuild the actual character bases to a higher pixel-voxel quality bar.

## Inputs from Phase 53d

- Voxelizer contracts
- Voxel rig contracts
- Spring-bone contracts
- External tool bridge commands
- Performance limits
- Reference asset intake rules

## Character base rules

- No hair.
- No facial hair.
- No baked identity traits.
- Neutral body.
- Neutral face plane only.
- Modular slots for future overlays.
- Tool attachment points included.
- Feet-center pivot.
- Scale compatible with generator profiles.

## Target body templates

```txt
character_base_body_a_neutral_bald_clean.vox
character_base_body_b_neutral_bald_clean.vox
character_base_player_neutral_bald_clean.vox
character_base_npc_average_neutral_bald_clean.vox
```

## Required quality improvements

- stronger silhouette,
- better head/neck/shoulder transition,
- shaped torso,
- clearer hips/legs,
- elbows/knees/hands/feet landmarks,
- modular attachment markers,
- clean front/side/back readability,
- pixel-voxel density used for anatomy and form, not random noise.

## Do not do in Phase 53e

- final clothing,
- final hair,
- final beards,
- final animation system,
- final runtime voxelizer,
- complex dynamic bone simulation.
