# Mobile LAN Editor

The mobile editor is served by the same LAN web editor server as the desktop web editor.

Recommended URL forms:

```text
http://<pc-lan-ip>:8787/
http://<pc-lan-ip>:8787/?mode=mobile
http://<pc-lan-ip>:8787/?mode=tablet
http://<pc-lan-ip>:8787/?mode=pixel
```

Mobile goals:

- map-first layout;
- large touch targets;
- bottom tool drawer;
- quick undo/redo/save;
- atlas picker mode;
- pixel editor mode;
- no separate APK requirement initially.

Later, this can be upgraded into an installable PWA.
