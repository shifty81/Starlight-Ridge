# Biomes

The canonical 12 biomes are:

1. Coastal Forest
2. Temperate Woodland
3. Meadow / Plains
4. Farmland / Cultivated
5. Marsh
6. Swamp
7. Taiga / Pine Forest
8. Tundra
9. Alpine / Snowfield
10. Arid Scrub
11. Desert / Dune
12. Volcanic / Ashlands

Biomes control material choices, weather, vegetation, liquids, snow behavior, and worldgen placement rules.
