# Content Schema Documentation

This folder describes the Phase 52 content contracts. Example RON files are provided under `content/contracts/phase52_examples/`.

Core contracts:

- Biome
- Material
- Liquid
- Weather
- Season
- Map layer
- VOX asset
- Worldgen preset
- Editor help metadata

The immediate Phase 52a implementation should convert these into Rust structs, RON loaders, and validators.
