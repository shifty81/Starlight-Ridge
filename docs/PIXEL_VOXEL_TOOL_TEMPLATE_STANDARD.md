# Pixel-Voxel Tool Template Standard

Generated tools are hero-detail starting points, not final production art.

Each tool should include:

- strong silhouette
- readable grip/handle zone
- clear working end
- material breakup
- hand attachment marker or pivot

Initial generated targets:

```txt
Hoe: 96 x 26 x 18
Pickaxe: 92 x 38 x 22
Axe: 88 x 34 x 24
Watering Can: 64 x 44 x 36
Sword: 90 x 20 x 16
```
